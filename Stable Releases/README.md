# Stable Releases

This folder contains all officially released versions of Operating Environment for Linux. Each version is stored in its own subfolder with the complete source code and a README describing the features and changes introduced in that release.

## Versions Included

- **v1.0** – First public release: Tic‑Tac‑Toe, Quiz, Calculator, Temperature Converter.
- **v2.01** – Second release: added linear equation solver, bootscreen, faster installer, single login.
- **v3.0** – Third release: difficulty levels in TicTacToe, registry editor, cheat codes, new quiz questions.
- **v4.05.08** – Fourth release: quadratic equation solver, settings app, user account improvements, Kelvin support, and more.

See each subfolder for detailed changelogs.
